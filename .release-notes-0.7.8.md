# TrueAsync PHP 0.7.8 (php 8.6)

## Highlights

- **Two use-after-free crashes fixed in the reactor and thread-pool internals** —
  including the intermittent rotation SIGSEGV left as a known follow-up in 0.7.7's
  reload() hardening.
- **`HttpServer::reload()` hang fixed** — a retiring sync-mode worker could block
  reload forever if the bootloader left a long-lived coroutine running.
- **Opcache ZTS leak fixed** — a class-linking race under threaded SAPIs leaked one
  linked class copy per race, found while investigating memory growth across
  worker reloads.

## Async extension / engine (php-async v0.7.7, php-src trueasync-0.7.7)

No ABI change — still TrueAsync ABI v0.23.0.

### Fixed
- **#176 Nested closures shared across worker threads → cross-thread `run_time_cache`
  use-after-free (SEGV).** A transferred closure's nested closures stayed shared with
  the persistent snapshot, so instantiating one on multiple workers raced on its
  run-time cache. The load path now deep-copies the op_array per worker when it
  carries nested definitions. This was the root cause of the intermittent rotation
  SIGSEGV noted as a known follow-up in 0.7.7 (true-async/server#102).
- **#173 Reactor: use-after-free when an IO request is disposed while its libuv
  operation is still in flight** (intermittent access violation on Windows process
  teardown). Closes UAF paths in armed stream reads, in-flight `uv_write`/fs ops,
  FILE-backed io, and not-yet-closed stream/UDP io.
- **`ThreadPool`: a retiring sync-mode worker could hang `HttpServer::reload()`
  forever** if the bootloader left a long-lived coroutine (e.g. a DB-pool
  healthcheck timer) running in the worker's main scope. The worker now
  force-starts graceful shutdown before its drain in that case.
- **opcache: free transient linked class copy on inheritance-cache race (ZTS).**
  In a multi-threaded (ZTS) SAPI, two threads could link the same immutable class
  concurrently and leak one linked copy; the losing thread's working copy is now
  freed on the cache-HIT path in `zend_accel_inheritance_cache_add()`.

## Installation

### Windows (PowerShell)

```powershell
irm https://raw.githubusercontent.com/true-async/releases/master/installer/install.ps1 | iex
```

To also install FrankenPHP non-interactively:

```powershell
$env:INSTALL_FRANKENPHP="true"; irm https://raw.githubusercontent.com/true-async/releases/master/installer/install.ps1 | iex
```

### Docker

```bash
docker pull trueasync/php-true-async:0.7.8-php8.6
docker pull trueasync/php-true-async:latest
```
