# TrueAsync PHP 0.7.13 (php 8.6)

Supersedes 0.7.12, whose Windows build failed — `src/grpc/grpc.c` included POSIX
`<strings.h>` unconditionally, which MSVC does not ship. This release carries the
fix (server v0.10.1) along with the same server delta as 0.7.12.

## Highlights

- **Windows build fixed.** The gRPC translation unit now guards `<strings.h>` with
  `#ifndef PHP_WIN32` like the rest of the tree, so the MSVC `fatal error C1083` is gone
  and Windows artifacts build again.
- **Correctness & interop across the HTTP stack.** Duplicate request headers are combined
  per RFC 9110 §5.3 on HTTP/1.1, HTTP/2 and HTTP/3 (`Cookie` joined with `"; "`, others
  with `", "`); WebSocket permessage-deflate honours the client's `server_max_window_bits`
  offer (RFC 7692).
- **Hardening & a gRPC streaming fix.** The inbound WebSocket message FIFO is bounded
  (8× `max_message_size`, close 1013 on overflow); an already-complete request is no longer
  upgraded to streaming — a deadlock that could park a gRPC handler until the connection died.

This is a server-only update (php-src, php-async and the ABI are unchanged).

## Components (php-src trueasync-0.7.8, php-async v0.7.10, server v0.10.1)

**No ABI change** — TrueAsync ABI stays at v0.24.0.

### Fixed
- **server: Windows build broken by the gRPC code** — `<strings.h>` included
  unconditionally (`fatal error C1083` under MSVC); now guarded with `#ifndef PHP_WIN32`.
- **server: duplicate request headers combined per RFC 9110 §5.3** across H1/H2/H3.
- **server: WebSocket permessage-deflate `server_max_window_bits` negotiation (RFC 7692).**
- **server: inbound WebSocket message FIFO bounded** (8× `max_message_size`) → close 1013.
- **server: an already-complete request is no longer upgraded to streaming** (gRPC deadlock).
- **server: reactor-spawned listeners get their own per-listener counters slice.**

### Performance
- **server: HTTP/3 reactor-mode request bodies built persistent from the first byte**; the
  wire body is adopted instead of re-copied; **gRPC `readMessage` reassembly compaction
  amortized**; assorted transport hot-path micro-cleanups.

### Changed
- **server: configure fails fast on a non-ZTS (NTS) PHP** — the threaded worker pool
  requires ZTS by design.
